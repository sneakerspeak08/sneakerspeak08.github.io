Website Evaluations

Pioneer DJ

Well built website for buying industry standard DJ equipment.

  • Contrast: Uses high contrast text that pairs well with the background, making the content readable.
  • Repetition: Consistent elements, fonts, and colors are used throughout this site.
  • Alignment: Content is aligned properly, and the layout is clear and concise.
  • Proximity: Related items are all grouped together, making it easy for the user to navigate through categories.

Pioneer DJ's website follows the CRAP principles effectively.

Craigslist (Charlotte)

Ad/markerplace website with an outdated and cluttered design.

  • Contrast: This website lacks contrast, making it hard to read in some areas.
  • Repetition: Inconsistent design elements across the site.
  • Alignment: Content is poorly aligned, which can cause confusion for some users.
  • Proximity: Elements are not organized, causing a cluttered look.

Craigslist Charlotte does not follow CRAP principles, resulting in a poor user experience. Its cluttered and outdated.
